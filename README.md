# 🛡️ AUDIT AI — Financial Transaction Forensics

Audit AI is an intelligent ledger forensics application designed to automate corporate transaction auditing and anomaly detection. Ingesting financial spreadsheet logs, the platform leverages advanced AI heuristics to instantly tag duplicate invoices, detect round-number withdrawals, and flag category-based risk vectors. It generates professional executive audit reports and presents visual timeline charts to give forensic auditors complete operational control.

---

## 🚀 Features

* **Forensic Pipeline Deep-Scan**: Ingests transaction logs to automatically classify categories and detect risk profiles in under 5 seconds.
* **Dynamic Anomaly Speedometer**: Displays a color-coded gauge speedometer mapping overall ledger compliance risk from green (Low) to yellow (Medium) to red (High) with counts.
* **Transaction Timeline Line Chart**: Interactive point-by-point line chart representing transaction flow with flagged anomalies highlighted as large red warning indicators on a sleek dark-themed timeline grid.
* **AI Chat Assistant**: Persistent secure chat interface leveraging actual spreadsheet data context to answer duplicate counts, peak outlier spikes, and strategic pre-audit advice.
* **Professional PDF Exporter**: Instant download of multi-page PDF audit reviews, featuring key metrics, anomaly tables, and priority remediation lists.
* **Audit History Lifecycles**: Locally cached index database tracking past audit runs and risk scores for secure comparative analysis.

## 🧠 How the AI Pipeline Works

1. **Step 1 — Classify**: Parses raw spreadsheet rows and executes category-matching validators to resolve unclassified entries and align general ledger fields.
2. **Step 2 — Detect Anomalies**: Scans files using statistical spike standard deviation metrics to locate duplicate invoices, round withdrawals ($500/$1000 limits), and entry bookkeeping errors.
3. **Step 3 — Generate Risk Score**: Combines severity indicators through a weighted compliance index formula to calculate a single 0-100% risk rating speedometer arc.

## 🚀 How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/AashishDhungana/audit-ai.git
cd audit-ai
```

### 2. Install dependencies
```bash
npm install
```

### 3. Spin up the development server
```bash
npm run dev
```

### 4. Compile static assets for production
```bash
npm run build
```
